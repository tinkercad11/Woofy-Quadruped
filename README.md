# Woofy-Quadruped
Woofy is a friendly quadruped robot, meant for beginners who are just starting out. Its simplicity lies in the fact that it uses only four servos, having no joints in each of the legs, makeing it easy to code and run. That is what makes woofy a really beginner-friendly project. 
It is designed to have only 1DOF per leb, making it very easy to create new gaits. 

## Materials 
### For robot : 
MG90(Recommended)/SG90 servo  x 4
ESP8266 NodeMcu Board  x 1
PCA9685 servo control board  x 1
1.3 inch oled display  x 1
Miscellaneous(Perfboard, headers, jumper wires)
### For Battery Pack : 
18650 cells  x 2
Suitable buck Converter  x 1
Cell holder  x 1
